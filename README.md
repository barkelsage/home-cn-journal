# home-cn-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涵盖日常记录、信息整理或特定主题的展示，以静态文件形式存放，便于浏览和版本管理。

## 项目简介

home-cn-journal 是一个纯前端静态页面集合，不依赖后端服务或数据库。每个 HTML 文件均为独立页面，可单独访问。仓库本身不绑定特定站点或域名，仅作为页面归档与发布的载体。

## 目录说明

```
home-cn-journal/
├── index.html          # 首页或导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 资源文件夹（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：可选的主页，可提供页面索引或跳转链接。
- `pages/`：核心内容目录，每个 HTML 文件代表一个独立页面。
- `assets/`：存放页面所需的公共资源文件。

## 页面归档说明

- 所有页面均为纯 HTML，可直接在浏览器中打开。
- 页面间无强制依赖关系，每个文件均可独立运行。
- 归档内容可能随时间更新或补充，历史版本通过 Git 提交记录保留。
